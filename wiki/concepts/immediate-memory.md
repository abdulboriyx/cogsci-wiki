# Immediate Memory

## Definition
Immediate memory refers to the cognitive system that holds a small amount of information in an active, readily available state for a short period of time.

## Distinctions
- **Immediate Memory vs. Absolute Judgment:** Absolute judgment refers to the ability to distinguish different stimuli (limited by [bits](/wiki/concepts/bit.md)), while immediate memory refers to the ability to recall a sequence of items (limited by [chunks](/wiki/concepts/chunking.md)).
- **Span of Immediate Memory:** The number of items an individual can correctly recall immediately after their presentation. Miller famously estimated this to be 7 ± 2.

## Interdisciplinary Uses
- **Neuroscience:** Studied in relation to the prefrontal cortex and other brain regions involved in short-term storage.
